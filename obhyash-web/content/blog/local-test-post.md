# এটি একটি লোকাল পোস্ট

এই পোস্টটি সরাসরি `content/blog` ফোল্ডার থেকে লোড হচ্ছে, নোশন থেকে নয়।

## সুবিধা

- দ্রুত লোড হয়
- গিটহাবে হিস্টরি রাখা যায়
- নোশন API এর উপর নির্ভরশীলতা কমে

> উভয় সিস্টেম একত্রে চমৎকারভাবে কাজ করছে!
